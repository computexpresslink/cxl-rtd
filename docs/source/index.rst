Welcome to the Linux CXL documentation
======================================

**Compute Express Link** (CXL) is a novel cache-coherent interconnect for CPUs, Memory Expansion and Accelerators. It is guided by the `CXL Consortium <https://computeexpresslink.org/>`_, forming an open standard across the industry, thereby facilitating its development and adoption.

.. note::

   This project is under active development.
